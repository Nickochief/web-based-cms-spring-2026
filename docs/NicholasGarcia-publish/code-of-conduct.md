## Code of Conduct

Acme Corp expects all employees to treat each other with respect and professionalism.

### Workplace Behavior

- Treat all colleagues with respect.
- Communicate professionally in all work settings.
- Report any misconduct to HR immediately.

### Confidentiality

- Do not share internal company information outside of Acme Corp.
- Keep all client data private and secure.
