## API Settings

This page contains the API configuration settings for the Acme Corp development environment. These settings are for internal use by engineers and developers only.

### Configuration

- API Version: v4.2.1
- Enable Caching: true
- Max Users: 500
- Admin Email: <sysadmin@acme.com>
- Feature Flags: beta_dashboard, dark_mode

### Feature Flags

Feature flags allow the team to enable or disable features without deploying new code. The following flags are currently active:

- **beta_dashboard**: Enables the beta version of the admin dashboard.
- **dark_mode**: Enables dark mode for the user interface.

For questions about API settings, contact the system administrator at <sysadmin@acme.com>.
