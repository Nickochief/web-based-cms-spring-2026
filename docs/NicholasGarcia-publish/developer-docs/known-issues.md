## Known Issues

This page documents known issues with Acme Corp's systems and their current workarounds. If you encounter a new issue, follow the reporting process outlined below.

### Current Issues

- **Dark Mode**: Dark mode does not work when you click it twice. If this happens, refresh the page to reset it.
- **Legacy Login**: The legacy login portal is no longer active. Do not link to or use the old intranet login.

### Reporting an Issue

If you encounter a new issue, report it to your manager immediately. Your manager will forward the details to IT Support, who will provide a fix or workaround.
